# On-chain Data Lab

🔍 A personal project for exploring blockchain data using Dune SQL.

Author: vb810195

## 📌 Goals

- Practice SQL-based on-chain analysis using platforms like Dune.
- Build dashboards for common DeFi protocols (e.g., Uniswap, Aave, meme coins).
- Record weekly learning progress and observations.

## 📁 Folder Structure

- `queries/` — SQL practice queries organized by protocol.
- `dashboards/` — Markdown descriptions of dashboards and metrics.
- `reports/` — Weekly summaries and insights.
- `utils/` — Common reusable SQL snippets and JSON parsing helpers.
